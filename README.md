
# 苍穹外卖后端技术栈

**核心框架**: Spring Boot 2.7.x + Spring MVC + Spring Security  
**数据持久层**: MyBatis + Druid连接池  
**数据库**: MySQL 8.0 主数据库 + Redis缓存数据库  
**认证授权**: JWT无状态认证 + Spring Security权限控制  
**接口文档**: Swagger/Knife4j自动生成API文档  
**第三方集成**: 阿里云OSS文件存储  
**开发工具**: Maven项目管理 + Git版本控制
